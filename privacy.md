# Politique de confidentialité — Pelouse Sacrée

*Dernière mise à jour : avril 2026*

## 1. Collecte des données
Pelouse Sacrée collecte uniquement les données nécessaires
au fonctionnement de l'application :
- Adresse e-mail (identifiant de connexion)
- Nom d'utilisateur
- Ville (optionnel)
- Photo de profil (optionnel)
- Données de tournois et matchs créés

## 2. Utilisation des données
Vos données sont utilisées exclusivement pour :
- Gérer votre compte et votre session
- Synchroniser vos tournois entre appareils
- Afficher votre profil aux autres utilisateurs

## 3. Stockage
Vos données sont stockées de manière sécurisée via
Supabase (infrastructure hébergée en Europe).
Nous n'vendons ni ne partageons vos données avec des tiers.

## 4. Vos droits
Vous pouvez à tout moment :
- Modifier vos informations depuis l'application
- Supprimer votre compte en nous contactant

## 5. Contact
Pour toute question : pelousesacree@gmail.com
